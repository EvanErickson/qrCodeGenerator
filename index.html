QR Code Generator

QR codes are huge in China. When I was there, people used QR codes for everything, from paying for food at a restaurant, to posting them on flyers for upcoming shows. I believe that QR codes will gain popularity in the United States so I wanted to create a way for the common person to get involved.

Using the latest Javascript available, along with Canvas, I successfully created an algorithm that will generate a QR code for any url. The hardest part of the project was dealing with all the variables and “drawing” the QR code. Luckily, my friend David Shim was there to help me.
